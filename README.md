---

## 🖥 Run Locally

### Prerequisites:
- Node.js installed

### Steps:

1. Install dependencies  

npm install


2. Add your Gemini API key inside `.env.local`

GEMINI_API_KEY=your_api_key_here


3. Start development server  

npm run dev
